# Morning Voice Clock

## 1. デザイン・コンセプト

**「北欧モダン × 朝のエネルギー」**

- **クリーン & シンプル:** 余計な線や枠を排除し、情報の優先順位を明確にする（時刻が主役）。
- **直感的:** 説明不要で使い方がわかる、大きなボタンやアイコン。
- **ポップ:** 太めのタイポグラフィや、鮮やかで親しみやすい配色（ポップなブルー＆イエローや、明るいグリーンなど）をアクセントに使用。

---

## 2. 視覚的要素（UI）

### 🖼️ フルスクリーン表示

- **巨大なデジタル時刻:** 画面中央にドカンと配置。フォントは視認性が高く、少し丸みのある太めのサンセ Serif（FuturaやNoto Sans Boldなど）を採用。
- **カラーパレット:**
  - **背景:** 清潔感のあるオフホワイト、または落ち着いたネイビー。
  - **時刻:** 視認性の高い黒、あるいはポップなイエロー。
- **レイアウト:**
  - 画面の9割を「時刻」が占める。
  - 設定（読み上げ間隔など）は、隅にある小さなアイコンをタップした時だけ表示される「引き算の美学」。

---

## 3. 機能的要素（UX）

### 🔊 オート読み上げ（聴覚）

- **インターバル通知:** 「7:15 です」「7:16 です」と、設定した間隔で淡々と、しかし明瞭な音声で通知。
- **音声トーン:** 機械的すぎず、朝を邪魔しないフラットで清潔感のある合成音声。

### ⚙️ 最小限の操作

- **スタート/ストップ:** 画面のどこかをタップ、あるいは大きな「再生/停止」アイコン一つで切り替え。
- **間隔設定:** 「1分 / 3分 / 5分」といった選択肢が並ぶだけのシンプルなトグル。

---

## 4. 利用体験のイメージ

1. **起動:** アプリを開くと、清潔感のある明るい画面に「07:10」と大きく表示されている。
2. **開始:** 「▶️」ボタンを一つ押すだけで、自動読み上げがスタート。
3. **洗顔:** 目を閉じている間、実用的で無駄のない音声が「7時11分です」と告げる。
4. **確認:** 顔を上げて鏡越しにスマホを見ると、ポップなカラーの巨大な数字が目に飛び込んできて、瞬時にあと何分あるか把握できる。

---

## 5. 技術スタック

- **Framework**: Next.js (Page Router, SSG export mode)
- **UI**: shadcn/ui
- **Styling**: Tailwind CSS 4 + tailwind-variants (tv)
- **Language**: TypeScript
- **Runtime**: Node.js 24 (required)
- **Package Manager**: pnpm >=10 (npm/yarn/bun forbidden)
- **Icons**: lucide-react
- **Notifications**: sonner (toast)
